# 🎣 Fishy Fishy!

A kid-friendly, two-player arcade fishing game — a web remake of a redemption
arcade cabinet. Two boats with swinging hooks; time the swing, drop the line,
and reel in whatever it snags. Deeper fish are worth more, the seafloor
**King Crab** and **Diamond** are the jackpots, and the junk row is good for a
laugh ("You caught an old Boot!").

It's a single `index.html` file — no build step, no dependencies.

## Play

- **iPad / touch:** tap a **boat** to cast that boat's line. Left half = Player 1,
  right half = Player 2.
- **Computer:** **Space** = left boat (P1), **Enter** = right boat (P2). `M` mutes.

The first press starts a shared **3‑2‑1‑GO!** countdown and a 60‑second race.
When the clock runs out, the higher score wins. Tap a boat to play again.

## Run it locally

Just open `index.html` in any browser.

## On an iPad

Open the GitHub Pages URL, then **Share → Add to Home Screen** for a
full‑screen, chrome‑free app.
